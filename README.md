# 電商網站 - Nutrition
> 此SPA專案作品為虛擬購物商店，銷售主題為有機蔬果，並包含前台以及後台。

### 簡介

前台功能 : 

* 商品展示(產品分類、單一產品詳細資訊)
* 購物車、優惠券及模擬簡易結帳
* 商品收藏清單
* 站內產品搜尋
* 切換瀏覽方式、商品排序

後台功能 : 

* 產品管理(新增、刪除、修改)
* 優惠碼管理
* 訂單列表管理
